既然我们编程的目的是为了控制计算机能够像人脑一样工作，那么人脑能做什么，就需要程序中有相应的机制去模拟。人脑无非是数学运算和逻辑运算，对于数学运算就是加减乘除。

## 1、算数运算符

计算机作为一种电子设备，主要功能之一就是执行数学计算并存储结果。在编程中，算术运算符扮演着基本数学运算的实现角色，它们用于执行最常见的数学运算，即四则运算及更多。

### （1）算术运算符

这些符号用于执行基本的数学计算。

| 运算符 | 描述 | 示例            |
| ------ | ---- | --------------- |
| +      | 加法 | `10 + 20 = 30`  |
| -      | 减法 | `10 - 20 = -10` |
| *      | 乘法 | `10 * 20 = 200` |
| /      | 除法 | `10 / 20 = 0.5` |
| //     | 整除 | `9 // 2 = 4`    |
| %      | 求余 | `9 % 2 = 1`     |
| **     | 指数 | `2 ** 3 = 8`    |

### （2）特殊用例和注意事项

**字符串的重复**：在Python中，`*`运算符可用于字符串，表示重复拼接字符串若干次。

```python
print("-" * 50)
# 输出: '--------------------------------------------------'
```

**数字型变量的计算**：

- 数字之间可以直接进行算术运算。
- 对于布尔类型，`True` 作为 `1` 计算，`False` 作为 `0` 计算。

**字符串的拼接**：

- 使用 `+` 运算符可以拼接字符串，创建新的字符串。

```python
first_name = "三"
last_name = "张"
print(first_name + last_name)
# 输出: '三张'
```

**类型兼容性**：

- 数字和字符串之间不能进行加法运算或其他算术运算，这会引发类型错误。

```python
first_name = "zhang"
x = 10
print(x + first_name)
```

## 2、比较（关系）运算符 

| 运算符 |                             描述                             |
| :----: | :----------------------------------------------------------: |
|   ==   | 检查两个操作数的值是否 **相等**，如果是，则条件成立，返回  True |
|   !=   | 检查两个操作数的值是否 **不相等**，如果是，则条件成立，返回 True |
|   >    | 检查左操作数的值是否 **大于** 右操作数的值，如果是，则条件成立，返回 True |
|   <    | 检查左操作数的值是否 **小于** 右操作数的值，如果是，则条件成立，返回 True |
|   >=   | 检查左操作数的值是否 **大于或等于** 右操作数的值，如果是，则条件成立，返回 True |
|   <=   | 检查左操作数的值是否 **小于或等于** 右操作数的值，如果是，则条件成立，返回 True |

> Python 2.x 中判断 **不等于** 还可以使用 `<>` 运算符
>
> `!=` 在 Python 2.x 中同样可以用来判断 **不等于**

## 3、逻辑运算符

在程序开发中，特别是在处理条件判断时，经常需要同时考虑多个条件。只有当这些条件以特定方式组合满足时，程序才执行后续的代码。这种条件组合是通过使用逻辑运算符来实现的。逻辑运算符允许你将多个条件逻辑地连接起来，形成更加复杂的条件判断语句。Python中提供了三种基本的逻辑运算符：**与（and）**、**或（or）**、**非（not）**。

### （1）逻辑运算符

| 运算符 | 逻辑表达式 | 描述                                                         |
| ------ | ---------- | ------------------------------------------------------------ |
| and    | x and y    | 如果x和y都为True，则返回True；否则返回False。                |
| or     | x or y     | 如果x和y中至少一个为True，则返回True；否则，如果都是False，则返回False。 |
| not    | not x      | 如果x为True，则返回False；如果x为False，则返回True。         |

### （2）and (逻辑与)

- **逻辑表达式:** 条件1 and 条件2
- **描述:** 两个条件必须同时为真，结果才为True。任一条件为False，则结果为False。

| 条件1 | 条件2 | 结果  |
| ----- | ----- | ----- |
| True  | True  | True  |
| True  | False | False |
| False | True  | False |
| False | False | False |

### （3）or (逻辑或)

- **逻辑表达式:** 条件1 or 条件2
- **描述:** 两个条件中只要有一个为真，结果就是True。只有当两个条件都为False时，结果才为False。

| 条件1 | 条件2 | 结果  |
| ----- | ----- | ----- |
| True  | True  | True  |
| True  | False | True  |
| False | True  | True  |
| False | False | False |

### （4）not (逻辑非)

- **逻辑表达式:** not 条件
- **描述:** 如果条件为True，则返回False；如果条件为False，则返回True。

| 条件  | 结果  |
| ----- | ----- |
| True  | False |
| False | True  |

### （5）使用场景

- **and 运算符**通常用于确保多个条件全部满足时才执行特定代码块。
- **or 运算符**用于在至少一个条件满足时执行代码块。
- **not 运算符**用于反转条件的逻辑状态。

通过合理应用这些逻辑运算符，可以构建复杂的逻辑表达式，使得程序逻辑更加灵活和强大。在编写条件语句时，理解并正确使用这些逻辑运算符是非常重要的，它们是控制程序流程的关键工具。

## 4、赋值运算符 

在Python编程中，赋值运算符扮演着将数据值分配给变量的核心角色。最基本的赋值运算符是 `=`，它用于将右侧表达式的值赋给左侧的变量。为了编写更加简洁高效的代码，Python还提供了一组与算术运算符结合的赋值运算符，这些运算符可以在执行算术操作的同时完成值的赋值。

### （1）赋值运算符

| 运算符 | 描述     | 示例        | 等效于       |
| ------ | -------- | ----------- | ------------ |
| =      | 简单赋值 | `c = a + b` | 无等效形式   |
| +=     | 加法赋值 | `c += a`    | `c = c + a`  |
| -=     | 减法赋值 | `c -= a`    | `c = c - a`  |
| *=     | 乘法赋值 | `c *= a`    | `c = c * a`  |
| /=     | 除法赋值 | `c /= a`    | `c = c / a`  |
| //=    | 整除赋值 | `c //= a`   | `c = c // a` |
| %=     | 求余赋值 | `c %= a`    | `c = c % a`  |
| **=    | 指数赋值 | `c **= a`   | `c = c ** a` |

### （2）注意事项

- **无空格规则：** 在使用赋值运算符时，确保运算符两侧不要有空格。例如，`c+=a` 是正确的，而 `c += a` 也是可以接受的，但在某些编程风格中，保持运算符周围无空格被视为最佳实践。
- **代码简化：** 这些赋值运算符有助于简化对变量的操作，使代码更加简洁明了。例如，`c += a` 能够有效地替代更冗长的 `c = c + a`，同时保持代码的可读性和清晰性。
- **适用场景：** 这些运算符尤其在需要对一个变量反复进行更新的情况下非常有用，如在循环中累加或递减数值，或者对变量进行连续的算术运算。

通过合理使用赋值运算符，你可以编写出更加高效和易于维护的代码。这些运算符不仅简化了代码，还帮助程序员快速实现复杂的数据处理逻辑。

## 5、运算符的优先级 

在Python中，就像在数学中一样，不同运算符有不同的优先级。这意味着在表达式中，高优先级的运算符会先被计算，而低优先级的运算符后计算。了解这些规则可以帮助你编写更有效的代码，并避免由于误解运算顺序而导致的错误。

### （1）基本规则

- **乘除优先于加减**：首先进行乘法、除法、取余和整除运算，然后执行加法和减法。
- **同级运算符**：在同一优先级的运算符中，计算顺序通常是从左到右。
- **使用括号**：可以使用括号`()`来改变运算的顺序，括号内的运算会优先执行。

### （2）运算符优先级表

以下是按优先级从高到低排列的运算符列表：

|          运算符          | 描述                   |
| :----------------------: | :--------------------- |
|            **            | 幂（最高优先级）       |
|         * / % //         | 乘、除、取余数、取整除 |
|           + -            | 加法、减法             |
|        <= < > >=         | 比较运算符             |
|          == !=           | 等于、不等于           |
| = %= /= //= -= += *= **= | 赋值运算符             |
|        not and or        | 逻辑运算符             |

### （3）逻辑运算符的优先级

逻辑运算符`not`、`and`、`or`的优先级依次增加：

- `not`具有最高的优先级
- `and`的优先级低于`not`
- `or`的优先级最低

### （4）示例分析

考虑下面两个表达式，思考它们的输出：

1. `(True or False) and False`
2. `True or False and False`

答案是，这两个表达式的结果不同。第一个表达式的结果是`False`，而第二个表达式的结果是`True`。这是因为`and`运算符的优先级高于`or`，所以第二个表达式实际上等价于`True or (False and False)`，因此结果为`True`。

### （5）编码建议

即使你确信了解优先级规则，使用括号来明确表达式的意图仍然是一种好的做法。这不仅可以避免潜在的错误，还可以使代码更易读，更容易理解。在包含多个`and`和`or`的复杂表达式中，明确地使用括号来指示运算的顺序总是一个明智的选择。

## 6、身份运算符

在Python中，身份运算符包括`is`和`is not`，它们用于比较两个对象的内存地址，即它们是否是同一个对象，而不是比较值的相等性。每个对象在Python中都有一个唯一的身份标识，可以通过`id()`函数获取。

### （1）`is`运算符

**用途**：检查两个对象是否是同一个对象，即它们在内存中是否有相同的地址。

**示例**：

```python
x = [1, 2, 3]
y = x  # y指向x相同的对象

print(x is y)  # 输出: True，因为x和y指向内存中同一个对象
```

### （2）`is not`运算符

**用途**：检查两个对象是否不是同一个对象，即它们在内存中是否有不同的地址。

**示例**：

```python
a = [1, 2, 3]
b = [1, 2, 3]

print(a is not b)  # 输出: True，因为a和b虽然值相同，但指向内存中不同的对象
```

### （3）`is`与`==`的区别

`is`：比较两个变量引用的对象是否为同一个（即比较对象的身份）。

`==`：比较两个变量引用的对象的值是否相等。

```python
a = [1, 2, 3]
b = [1, 2, 3]
print(b is a)  # 输出: False，因为b和a引用不同的对象
print(b == a)  # 输出: True，因为b和a引用的对象值相同
```

### （4）与`None`值的比较

在Python中，`None`是一个单例，推荐使用`is`来判断变量是否为`None`。这样可以确保比较的严格性和准确性。

**示例**：

```python
def example():
    return None


if example() is None:
    print("Result is None")
```

**注意**：即使你可以通过自定义`__eq__`方法使得对象在使用`==`时表现为等于`None`，也应当使用`is`来进行判断。

身份运算符是Python中用于比较对象身份的重要工具，它们在处理单例、缓存或优化等场景下特别有用。理解`is`与`==`的区别对于编写清晰和正确的代码至关重要。在判断变量是否为`None`时，始终使用`is`以避免意外行为。

## 7、成员运算符

成员运算符在Python中扮演着至关重要的角色，主要用于判断一个值是否为某个序列（如列表、元组、字符串、集合）或字典的成员。Python提供了两种成员运算符：`in`和`not in`。

### （1）`in`运算符

使用`in`运算符可以验证指定的值是否是容器中的一个元素。

**示例：列表**

```python
my_list = [1, 2, 3, 4, 5]
print(3 in my_list)  # 输出: True，3是列表元素
print(6 in my_list)  # 输出: False，6不是列表元素
```

**应用于不同类型的容器**

```python
my_string = "Hello, world!"
print("world" in my_string)  # 输出: True，字符串中包含"world"

my_tuple = (1, 2, 3)
print(2 in my_tuple)  # 输出: True，元组中包含2

my_set = {1, 2, 3}
print(4 in my_set)  # 输出: False，集合中不包含4

my_dict = {"a": 1, "b": 2}
print("b" in my_dict)  # 输出: True，字典的键中包含"b"
```

### （2）`not in`运算符

`not in`与`in`相反，用于判断一个值是否不在容器中。

**示例：列表**

```python
my_list = [1, 2, 3, 4, 5]
print(6 not in my_list)  # 输出: True，6不是列表元素
print(3 not in my_list)  # 输出: False，3是列表元素
```

成员运算符特别适用于在不需要完整遍历容器的情况下快速检查元素是否存在，提高代码的简洁性和效率。无论是在编写条件语句、循环、函数还是其他数据操作时，正确使用`in`和`not in`运算符都能使代码更加直观易懂。
