## 一、定义及创建

### 1. 程序中的变量作用

程序的执行本质上是一连串状态的变化。这些变化表现为数据的转换、更新和管理，而变量正是这些动态数据的载体。变量使得程序能够记录和修改信息，从而响应不同的输入和环境条件。例如：

- 一个游戏中的英雄等级从1升级到10，这个等级变化可以通过变量来跟踪。
- 在一个模拟环境中，一个僵尸的存活状态可以从`True`（活着）变更为`False`（被消灭）。
- 用户的姓名可能最初被设定为"Albert"，后来更新为"AlbertMa"。

通过变量，程序不仅可以处理当前的数据状态，还可以根据逻辑和用户交互改变这些数据，实现动态的功能。

### 2. 定义及创建变量

在Python中，变量是一个存储数据值的标识符。定义变量时，不需要显式声明其类型。Python解释器会根据赋给变量的值自动确定变量的类型。变量的创建发生在首次为其赋值时，这一点与某些其他编程语言不同，其中可能需要显式声明变量类型或先声明后赋值。

变量赋值使用等号（`=`），等号的左边是变量名，它是数据的标识；等号的右边是要存储在变量中的值。

**语法结构**

```python
variable_name = value
```

**示例**

```python
level = 1       # 定义一个名为level的变量，初始值为1
level = 10      # 更新变量level的值为10
is_alive = True # 定义一个名为is_alive的布尔变量，初始值为True
is_alive = False # 更新变量is_alive的值为False
name = "Albert" # 定义一个名为name的字符串变量，初始值为"Albert"
name = "AlbertMa" # 更新变量name的值为"AlbertMa"
```

一旦定义，就可以在程序中自由使用这些变量，以引用它们的值或更新它们的值。变量名在首次出现时定义变量，之后出现的同名变量则指向已定义的变量，并可以用来访问或修改其值。

## 二、变量的类型

当在内存中创建一个变量时，它确实涉及到几个重要的组成部分，这些部分让变量在程序中起到了存储和引用数据的作用。在 Python 这样的动态类型语言中，变量的处理有其特定的方式。

### 1. 变量的组成

**变量的名称**:

- 这是变量的标识符，用于在代码中引用变量。通过变量名，程序可以访问或者更新存储的数据。

**变量保存的数据**:

- 这是变量指向的实际值。它可以是数字、字符串、列表、字典、对象等任何数据类型的值。

**变量存储数据的类型**:

- 即使在Python中定义变量时不需要显式声明其类型，变量仍然关联着一个特定的数据类型。这个类型是根据变量被赋予的值动态确定的。

**变量的地址（标识）**:

- 每个变量都存储在内存的一个位置，这个位置通过地址来标识。可以通过Python的`id()`函数获取变量的内存地址。

### 2. Python中的变量和类型

Python是一种动态类型语言，这意味着你在定义变量时不需要显式指定数据类型。Python解释器会根据变量赋值的内容自动确定变量的类型。这种特性增加了语言的灵活性，使得Python代码更加简洁易读。

使用`type()`函数，你可以查询任何变量的数据类型，这是检查变量中存储数据类型的一种快捷方式。例如：

```python
name = "Alice"
print(type(name))  # 输出: <class 'str'>

age = 30
print(type(age))  # 输出: <class 'int'>
```

在这些例子中，`name`变量被自动识别为字符串（`str`），而`age`变量被识别为整数（`int`）。这展示了Python如何根据赋给变量的值自动推导数据类型。

## 三、变量的输入

在Python中接收和处理输入是一项基本技能，适用于多种不同的应用场景。

### 1. 从命令行接收输入

使用`input()`函数是一种交互式的方式，允许程序在运行时接收用户输入。这种方法适用于需要用户实时输入数据的情况。例如，当需要用户输入配置信息、进行选择或提供即时数据时。

```python
name = input("请输入您的名字：")
print(f"您的名字是：{name}")
```

这种方法的优点是简单易用，但它使程序的执行暂停，等待用户输入，这在自动化脚本或后台运行的程序中可能不是最佳选择。

### 2. 使用命令行参数

`sys.argv`是一个列表，其中包含命令行参数，即运行Python脚本时从终端输入的参数。`sys.argv[0]`是脚本名称，后续项是传递给脚本的参数。

```python
import sys

if len(sys.argv) > 1:
    name = sys.argv[1]
    print(f"您的名字是：{name}")
else:
    print("请提供一个名字作为参数。")
```

命令行参数适合于那些需要从外部接收配置或数据的程序，特别是在批处理或自动化脚本中非常有用。

### 3. 从文件中读取输入

将数据存储在文件中，并在程序运行时读取，是自动化处理大量数据的常用方法。Python通过内置的文件处理功能，使得从文件中读取数据变得非常简单。

```python
with open("input.txt", "r") as file:
    name = file.readline().strip()
    print(f"您的名字是：{name}")
```

这种方法非常适合处理大量数据，或当数据提前准备好并存储在文件中时。

### 4. 通过环境变量获取输入

环境变量是操作系统中存储的一种变量，可以被运行在该操作系统上的所有程序访问。Python可以通过`os.environ`或`os.getenv()`访问这些变量，这对于从外部环境获取配置信息非常有用。

```python
import os

name = os.getenv("USER_NAME", "Guest")
print(f"您的名字是：{name}")
```

这段代码尝试获取名为`USER_NAME`的环境变量的值。如果变量不存在，它将默认使用`'Guest'`。

## 四、变量的输出

在Python中，格式化输出是一种常见且强大的功能，它允许程序员以可读且结构化的方式展示数据。Python提供了多种方式来格式化字符串，使其包含变量、表达式的值或字面量文本。下面是对提到的内容的优化和完善：

### 1. 使用占位符（%）

这种古老的字符串格式化方法借鉴自C语言，使用特定的占位符来代表不同类型的数据，如字符串（`%s`）、整数（`%d`）、浮点数（`%f`）等。在占位符中，可以指定宽度、精度和对齐方式等。

```python
name = "Tom"
age = 20
print("My name is %s and my age is %d years old." % (name, age))
# My name is Tom and my age is 20 years old.
```

这种方法虽然有效，但已不是推荐的首选方式，因为它在一些情况下可能会导致代码的可读性降低，特别是在处理复杂的格式化时。

### 2. 使用format函数

`format`方法是一种更现代的字符串格式化方法。它使用大括号`{}`作为特殊的标记，可以包含一个可选的格式说明符。这种方法提高了格式化的灵活性和可读性。

```python
name = "Tom"
age = 20
print("My name is {} and my age is {} years old.".format(name, age))
# My name is Tom and my age is 20 years old.
```

`format`方法的一个优点是可以重复使用参数，还可以使用关键字参数，使得格式化字符串更加清晰。

### 3. 使用f-string（格式化字符串字面量）

自Python 3.6起，f-string提供了一种快速且易读的方式来嵌入表达式的值于字符串字面量中。这通过在字符串前加上`f`或`F`，并将表达式放置在大括号中来实现。

```python
name = "Tom"
age = 20
print(f"My name is {name} and my age is {age} years old.")
# My name is Tom and my age is 20 years old.
```

f-string被认为是最快且最直观的字符串格式化方法，因为它允许直接在字符串中嵌入表达式。它不仅使代码更加简洁，而且执行速度也很快。

## 五、变量的引用

在Python中，变量的引用是指向内存中存储的数据值的链接。当我们创建变量并赋值时，变量名实际上就是对内存地址的引用，该地址存储着实际的数据。理解变量的引用机制对于高效且正确地编写代码至关重要，特别是在处理可变类型和不可变类型时。

### 1. 引用的概念

在Python中，所有的数据都是对象，变量是对这些对象的引用。当我们执行如下赋值操作时：

```python
a = 10
b = a
```

这里，`a` 是指向数值 10 的引用，而当我们将 `a` 赋给 `b`，`b` 也成为数值 10 的一个引用。在这个过程中，没有复制任何数据，只是增加了一个新的引用到原始对象。

### 2. 不可变类型与可变类型

- **不可变类型**: 数值、字符串、元组。这些类型的对象一旦创建，其内容不可更改。如果尝试修改，实际上会创建一个新的对象，并改变相应变量的引用。

```python
x = 5
y = x
x = 3
print(y)  # y 的值仍然是 5
```

在这个例子中，`y` 的值不会随 `x` 的改变而改变，因为整数是不可变的。当我们给 `x` 赋一个新值时，实际上是创建了一个新的整数对象，并改变了 `x` 的引用。

- **可变类型**: 列表、字典、集合。这些类型的对象可以在不改变内存地址的情况下更改其内容。

```python
a = [1, 2, 3]
b = a
a.append(4)
print(b)  # b 的输出为 [1, 2, 3, 4]
```

这里，`a` 和 `b` 都引用同一个列表。当我们修改这个列表时，通过任何一个引用都能看到这些更改。

### 3. 引用与函数

函数参数的传递也是通过引用进行的。这意味着如果你向函数传递了一个可变对象，那么函数内部对这个对象的任何更改都会反映到原始对象上。

```python
def modify(lst):
    lst.append(4)


my_list = [1, 2, 3]
modify(my_list)
print(my_list)  # 输出 [1, 2, 3, 4]
```

在这个例子中，`modify` 函数内部对传入列表的修改，影响了原始列表 `my_list`。

## 六、局部变量和全局变量

在Python中，理解局部变量和全局变量是编写清晰、有效代码的关键部分。这两种变量类型的主要区别在于它们的作用域，即变量存在和可以被访问的区域。

### 1. 全局变量

全局变量是在程序的顶层定义的，它们可以在代码的任何地方被访问。通常，全局变量用于整个程序中多个函数或模块共享的数据。

#### （1）定义全局变量

全局变量通常在脚本的最顶层定义，即所有函数和类的外部。

```python
global_var = "I am a global variable"

def my_function():
    print(global_var)  # 可以访问全局变量

my_function()
```

在这个例子中，`global_var` 是一个全局变量，可以在函数内部被访问和打印出来。

#### （2）修改全局变量

如果你需要在函数内部修改全局变量，需要使用 `global` 关键字。

```python
global_var = "I am a global variable"


def my_function():
    global global_var
    global_var = "Modified global variable"


my_function()
print(global_var)  # 输出: Modified global variable
```

在这个例子中，`my_function` 函数修改了全局变量 `global_var`。

### 2. 局部变量

局部变量是在函数内部定义的变量，只能在其被定义的函数内部访问。它们在函数被调用时创建，在函数执行完毕后销毁。

定义局部变量

```python
def my_function():
    local_var = "I am a local variable"
    print(local_var)  # 在函数内部打印局部变量


my_function()
# print(local_var)  # 这会抛出错误，因为 local_var 在这个范围内是不可见的
```

在这个例子中，`local_var` 只能在 `my_function` 内部被访问。

### 3. 注意事项

- **命名冲突**：当局部变量和全局变量同名时，局部变量会在其作用域内覆盖全局变量。
- **`global` 关键字**：如果你想在函数内部修改全局变量，必须使用 `global` 关键字。
- **可变类型注意事项**：如果全局变量是可变类型（如列表、字典等），即使不使用 `global` 关键字，你也可以在函数内部修改它的内容。但是，如果你尝试重新绑定该变量（例如，将其指向一个新的列表或字典），则必须声明为全局。

### 4. 使用建议

- **最小化全局变量的使用**：依赖全局变量可能会使得代码难以理解和维护。尽可能通过函数参数传递数据。
- **明确变量作用域**：清晰地了解每个变量在你的程序中的作用域可以帮助你避免许多常见的错误。
- **代码组织**：保持代码的结构清晰，尽量使得函数自包含，这样更容易理解和维护。
