## 一、python文件

Python 源程序就是**一个特殊格式的文本文件**，可以**使用任意文本编辑软件**做 `Python` 的开发，Python 程序的**文件扩展名**通常都是 `.py`

## 二、行与缩进

### 1. 行（Lines）

Python 语句通常逐行书写，每行代表一个单独的代码指令或逻辑部分。

### 2. 缩进（Indentation）

Python 使用缩进来定义代码块，而不是像其他语言那样使用花括号 `{}`。通常使用4个空格。不要混用空格和制表符（Tab），以避免缩进错误。

- **示例代码**

```python
if condition:
    # 缩进的代码块
    print("Condition is True")
else:
    print("Condition is False")
```

## 三、多行代码

当单行代码过长，无法在一行内完整写下时，可以将其分行书写，以保持代码的可读性。在 Python 中，可以使用反斜杠（`\`）来实现多行语句，例如：

```python
count = 3 + 4 \
        + 6 + 9 \
        + 10 + 11
```

在定义括号、方括号或花括号内的多行语句时，不需要使用反斜杠，Python 会自动将它们视为连续的一行。例如，定义一个多行列表：

```python
lst = [1, 3, 4,
       5, 5, 2,
       7, 9, 10]
```

## 四、命名规则

### 1. **命名基础**
**区分大小写**：Python 变量名区分大小写，`variable` 和 `Variable` 是两个不同的标识符。

**字母、数字和下划线**：标识符只能包含字母、数字和下划线（`_`），但不能以数字开头。例如，`variable1` 是合法的，`1variable` 是非法的。

### 2. **变量和函数命名**
**小写字母，单词间用下划线分隔**：使用小写字母，并使用下划线 `_` 分隔多个单词。

- 示例：`user_name`, `total_count`, `get_user_data`

### 3. **常量命名**
**全大写字母，单词间用下划线分隔**：常量通常使用全大写字母，并用下划线 `_` 分隔多个单词。

- 示例：`PI`, `MAX_CONNECTIONS`, `DEFAULT_TIMEOUT`

### 4. **类命名**
**首字母大写的驼峰命名法（PascalCase）**：每个单词的首字母大写，其他字母小写。通常用于类的命名。

- 示例：`UserProfile`, `DataAnalyzer`, `OrderProcessor`

### 5. **私有变量和函数**
**以单个下划线开头**：表示私有变量或私有方法，不应从模块外部直接访问。该约定仅用于提示作用，Python 本身不会限制访问。

- 示例：`_private_variable`, `_calculate_total`

**以双下划线开头**：表示类内私有变量或方法，Python 会进行名称重整（name mangling），使其不易在类外访问。

- 示例：`__private_method`

### 6. **模块和包命名**
**小写字母，单词间用下划线分隔（可选）**：模块名应使用小写字母，可以使用下划线分隔单词。包名通常为单个单词的小写字母。

- 示例：`utilities.py`, `data_processing.py`, `myproject`

### 7. **避免使用保留字**
**不要使用 Python 的保留字**：保留字是 Python 语言中有特殊含义的单词，不能用作标识符。常见保留字包括 `class`, `def`, `return`, `import`, `if`, `else` 等。

要查看所有保留字，可以使用以下代码：
```python
import keyword
print(keyword.kwlist)
```

### 8. **避免与内置函数名冲突**
**避免覆盖 Python 内置函数名**：例如，不要将变量命名为 `list`, `str`, `input` 等。这会导致内置函数不可用或产生意外行为。

- 错误示例：
  ```python
  list = [1, 2, 3]  # 不推荐，会覆盖内置的 list 函数
  ```

### 9. **PEP 8 风格指南**
**遵循 PEP 8**：这是 Python 官方推荐的编码风格指南，涵盖了命名规则、代码布局等各个方面。遵循 PEP 8 有助于保持代码的一致性和可读性。

## 五、Python 中的计数方法

常见的计数方法有两种，可以分别称为：

* **自然计数法**（从 `1` 开始）—— 更符合人类的习惯
* **程序计数法**（从 `0` 开始）—— 几乎所有的程序语言都选择从 0 开始计数

因此，在编写程序时，应该尽量养成习惯：**除非需求的特殊要求，否则计数都从 0 开始**
